About

flyUtils is intended to be a bunch of web-apps that would aid fly researchers in many routine tasks. And this is completely open source, hosted at github (https://github.com/roastedneutrons/FlyCross).

singleCross is meant for analyzing single-step drosophila crosses. The app is intended for both newcomers to fly research, and for experienced fly pushers. (However, it assumes the user is acquainted with standard drosophila nomenclature). When you are setting up crosses with multiple markers on different chromosomes, drawing a punnett square and figuring out phenotype clashes is a pain to do on pen and paper. This tool is intended to help in such situations.

Features in the pipeline:

  • Yield percentage of each progeny: Useful in multi-step crosses, if you have multiple genotypes that work for you, and want to know quickly
  • Markers to screen for: Info about which markers have to be screened to isolate your progeny from the rest of the population. Although this is easy to figure out from your cross, manually, its nice if everything you want is displayed in one place.
  • A proper fly-work API, so that other programs can easily interface with this.
  • Easier specification of constraints: Right now constraints have to be typed in. Some might find this annoying. So an optional way of clicking genes on the displayed genotype to specify constraints would be handy.
  • Auto-filling the constraints, balancers and markers, by extracting data from FlyBase
